Asta4D Sample Project - Index

Simple usages

  • Use Snippet for rendering dynamic page.
  • Inject values of the variable.(/{name}/{age})
    • name=foo, age=25
    • name=bar, age=20
  • Manipulate attribute values.
  • Template extension
  • Embed other template.
  • Localize Messages(i18n support).
  • Error handling
    • 404 exception by request handler
    • exception occurs at the request handler
    • exception occurs at the snippet

Complicated usages

  • Form flow
    • Single input step
    • Input, Confirm and Complete
    • Cascade and array field
    • Multiple input steps
    • Splitted input steps
  • Experience some type of Renderers.
  • Pass variables from outer Snippet to inner Snippet.
  • Use ContextData annotation.
